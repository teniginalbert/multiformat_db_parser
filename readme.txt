Задание работает и соответствует тз.

По умолчанию таблица загружается из файла database.db, лежащего в папке с исполняемым файлом.
Если его нет, то создаёт такой файл.

Для парсинга xml-файлов нужно выбрать директорию и нажать кнопку "парсинг xml-файлов".
Прогресс выводится в лог и отображается на прогресс-баре.

Таблицу можно изменять, удалять строки, очистить таблицу. 
Таблица по результату парсинга xml-файлов изменяется автоматически.
Изменения, вносимые в таблицу, не вносятся в бд (доп. задание).

Для экспорта в xml-файл нужно выбрать строку таблицы и нажать "сохранить xml-файл".
Файл создаётся в директории исполняемого файла.

Пасхалка в тз была найдена в процессе работы программы.
В папке тестовых xml-файлов не парсятся файлы notepad++ и и sublime.
В первом ошибка "Expected character data", файл кодирован не в заявленном UTF-8,
во втором - открывающий и закрывающий тег не совпадают.

Комплект сборки
Desktop QT 5.13.1 MinGW 32-bit

Этапы сборки
qmake: qmake.exe test_task.pro -spec win32-g++ "CONFIG+=debug" "CONFIG+=qml_debug"
Сборка: mingw32-make.exe -j4 in %output_directory%
